Preferred flows out of S-77 and S-308 are determined by using the following criteria:
  - At low flows (<300 cfs), the strcuture flow is used.
  - If all gates are closed, the flow is zero.
  - If the structure flow is greater then the below gage by more than 30%, then the
     downstream gage will be the preferred flow. This scenatio is likely due to
     vegetation clogging which causes the structure flow to be falsely high.
  - If there is positive head accross the structure, negative flows will not be used
  - USACE can force the preferred flows to be one gage or the other, overwriting the above rules
     (exception is if the gates are closed, then flow will be zero)
